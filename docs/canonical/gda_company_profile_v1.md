## 3. System of Systems

The enterprise runs as 3 integrated systems. **Value is created or destroyed at handoff points.**

```
       BD System  ─────┐ what is sold, how priced, how promised
                       │
   ┌──Resourced──┐  Operations System ──Measured──┐
   │             │  executes commitments          │
   │             │                                │
   │       Back Office System ──Promised──────────┘
   │       cash, controls, compliance, people
```

Every Envision record in the tool tags to which system handed it off. Audit handoffs surface where execution breaks. Partner records are exempt (we don't operate their handoffs, only observe their public posture).
