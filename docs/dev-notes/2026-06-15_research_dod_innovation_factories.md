# DoD / Military Innovation Organizations — FasTrac Signal Source Catalog
**Purpose:** Early-warning signal sensing for GDA Command / FasTrac need-sensing feature  
**Compiled for:** Envision (Alexandria, VA) — Government-contracting BI tool  
**Last updated:** June 2025  
**Coverage:** 60+ active organizations publishing tech solicitations, RFIs, prize challenges, CSOs, BAAs, and partnership opportunities prior to formal SAM.gov posting

---

## Tier Definitions
- **Tier 1:** Highest signal value — publishes regularly, broad industry engagement, primary CSO/SBIR/OTA vehicles. Monitor weekly.
- **Tier 2:** Secondary value — less frequent, narrower scope, or government-internal with occasional public signals. Monitor monthly.
